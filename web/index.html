API Reference

Endpoints

Submit Job via URL

POST /api/job/submit
Content-Type: application/json

{
  "videoUrl": "https://example.com/video.mp4",
  "silenceThreshold": -30,
  "silenceDuration": 0.5,
  "removeSilence": true,
  "generateCaptions": true,
  "captionStyle": "TikTok",
  "webhookUrl": "https://your-app.com/webhook"
}

Submit Job via File Upload

POST /api/job/upload
Content-Type: multipart/form-data

file: [video file]
silenceThreshold: -30
silenceDuration: 0.5
removeSilence: true
generateCaptions: true
captionStyle: TikTok
webhookUrl: https://your-app.com/webhook

Get Job Status

GET /api/job/status/{jobId}

Get All Jobs

GET /api/job/all

Cancel Job

POST /api/job/cancel/{jobId}

Download Processed Video

GET /api/download/output/{jobId}/{filename}
